## 🔹 Kubernetes - Deployment Rolling Update (Detailed)

apiVersion: apps/v1
kind: Deployment
metadata:
  name: codr-rollout
spec:
  replicas: 3
  strategy:
    type: RollingUpdate
    rollingUpdate:
      maxSurge: 2
      maxUnavailable: 1
  selector:
    matchLabels:
      app: rollout
  template:
    metadata:
      labels:
         app: rollout
    spec:
      containers:
         - name: app
           image: nginx

# Approvals

## 🔹 Jenkins Pipeline - Manual Approval

You will learn how to **pause pipeline until manual approval**.

pipeline {
    agent any
    environment {
        INSTITUTE_NAME = "codr-institute"
    }
    stages {
        stage('Deploy') {
            steps {
                input message: "Approve deployment for $INSTITUTE_NAME?"
                sh 'echo "Deployment approved and executed!"'
            }
        }
    }
}

## 🔹 GitLab CI - Manual Approval Job

You will learn how to **use manual jobs in GitLab CI for approvals**.

stages:
  - deploy

variables:
  INSTITUTE_NAME: "Codr-Institute" 

deploy:
  stage: deploy
  image: alpine:latest
  script:
    - echo "Ready to deploy $INSITUTE_NAME"
  when: manual



# Day 19 - Advanced Monitoring , Logging, and Security

## 🔹 Ansible - Setup Prometheus for Monitoring 

- name: Install Prometheus for Monitoring
  hosts: all
  become: yes
  tasks: 
    - name: Install Dependencies for Prometheus
      yum: 
        name:
          - prometheus
          - prometheus-node-exportor
    - name: Start Prometheus service
      service:
        name: prometheus
        state: started
        enabled: yes

## 🔹 Jenkinsfile - Integrating Prometheus Monitoring in Pipeline 

pipeline {
    agent any 
    stages {
        stage('Build') {
            steps {
                echo 'Building Docker image...'
                docker.build('codr-web-app')
            }
        }
        stage('Test') {
            steps {
                echo "Running Tests..."
            }
        }
        stage('Monitor Deployment') {
            steps {
                script {
                    sh 'kubectl apply -f kubernetes/deployment.yaml'
                    sh 'kubectl get pods --watch'
                }
            }
        }
    }
}
