# Artifacts Promotion

## 🔹 Jenkins Pipeline  - Pormotion Artifacts
You will learn how to **promote build artifacts from dev to stagging**.

pipeline {
   agent any
   environment {
      INSTITUTE_NAME = "Codr-Army"
   }
   stages {
      stage('Build') {
         steps {
            sh 'mvn clean package'
         }
      }
      stage('Promote') {
         steps {
            input message: "Promote artifacts to stagging?"
            sh 'echo "Promotion artifacts for $INSTITUTE_NAME"'
         }
      }
   }
}
